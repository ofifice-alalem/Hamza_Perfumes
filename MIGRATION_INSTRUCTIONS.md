# تعليمات تشغيل Migration الجديد

## لتطبيق التحديثات الجديدة على قاعدة البيانات:

1. افتح Command Prompt أو Terminal في مجلد المشروع
2. شغل الأمر التالي:

```bash
php artisan migrate
```

## التحديثات المضافة:

### 1. حقول جديدة في جدول perfume_prices:
- `bottle_price_regular` - سعر العبوة للعملاء العاديين
- `bottle_price_vip` - سعر العبوة لعملاء VIP

### 2. تحديثات في الواجهات:
- صفحة إضافة الأسعار (`/prices/create`)
- صفحة تعديل الأسعار (`/prices/{id}/edit`)

### 3. الميزات الجديدة:
- إمكانية تحديد سعر منفصل للعبوة الكاملة للعملاء العاديين
- إمكانية تحديد سعر منفصل للعبوة الكاملة لعملاء VIP
- نفس النظام المطبق على الأحجام المقسمة

## ملاحظات:
- الحقل القديم `bottle_price` سيبقى موجود للتوافق مع البيانات القديمة
- الحقول الجديدة اختيارية (nullable)
- يمكن ملء أحد الحقلين أو كليهما حسب الحاجة
